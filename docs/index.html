Clinical Trials Analysis R

Este repositório, desenvolvido por Gabriel Demetrios Lafis, oferece um sistema profissional para análise de ensaios clínicos utilizando a linguagem R. Inclui modelagem estatística avançada, análise de sobrevivência, visualizações interativas e relatórios automatizados para pesquisa clínica e farmacêutica.

Recursos Principais:

  • Análise Estatística Avançada
  • Análise de Sobrevivência
  • Análise de Eficácia e Segurança
  • Visualizações Regulamentares
  • Relatórios Automatizados (RMarkdown)
  • Dashboard Interativo (Shiny App)
